## Installation

Binar Car Rental requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
cd /{YOUR_PATH_FOLDER}/
npm install
npm run dev
```

## Building for production
```
npm run build
```
